# 🤖 Weather Pro Chatbot 🌦️

A modern chatbot built using Python and Streamlit that provides real-time weather updates and 5-day forecasts for any city.

---

## 🚀 Features

* 🌍 Get live weather for any city
* 🌡️ Temperature and wind speed
* ☀️ Weather icons (sun, rain, fog, etc.)
* 📅 5-day forecast
* 💬 Chat-style UI (like WhatsApp)

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 💻
* Open-Meteo API 🌦️

---

## 📦 Installation

```bash
pip install streamlit requests
```

---

## ▶️ Run the App

```bash
streamlit run weather_chatbot_pro.py
```
